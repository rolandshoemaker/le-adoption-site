Methodology

Built with Golang + CT, called ca-adoption-scanner, is on github. ^_^

Scan scope

This chart shows the number of certificates we extracted from CT logs to check, the number of DNS names extracted from those certificates to scan, and the duration of each scan.

Adoption information

This chart shows the percentage of DNS names that use their certificate as a percentage of the number of the total number of names scanned. It also shows the number of certificates that are used either by either none, some, or all their DNS names.

Feature usage

This chart shows the percentage of DNS names that support various TLS features or extensions. Currently this only tracks deployment of OCSP response stapling and the inclusion of SCT receipts in either the TLS handshake, or in a stapled OCSP response.

Negotiated cipher suites

This chart shows a breakdown of the cipher suites negotiated during successful TLS handshakes.

Scanned name issues

This chart shows the various rates of errors the scanner runs into while attempting to preform TLS handshakes with DNS names.
